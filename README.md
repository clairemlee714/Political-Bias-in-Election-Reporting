Project analyzing how four news outlets (NPR (left-leaning US), the New York Post (right-leaning US), AP (non-partisan US) and BBC (non-partisan international)) framed the 2020 U.S. Presidential Election between Donald Trump and Joe Biden. Using articles sourced from Media Cloud and scraped from each outlet's website, I'm examining coverage at two levels: general election articles and candidate-specific articles that exclude mentions of the opposing candidate. Applying NLP techniques (sentiment analysis, TF-IDF, topic modeling, and named entity recognition) to compare differences in tone, language, and thematic emphasis across outlets.
